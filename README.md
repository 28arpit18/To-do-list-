# To-do-list-
This script and code allows you to add, view, and remove tasks until you choose to exit.
➡️Key Concepts Used:
1️⃣​while True:: Creates a continuous loop so the program doesn't stop until you explicitly choose to exit.
2️⃣​list: Used (todo_list) to store the collection of tasks.
3️⃣​enumerate(): A very useful function that gives you both the index and the item while looping, making it easy to display tasks with numbers.
4️⃣​try-except: Used in the removal section to prevent the program from crashing if a user types something that isn't a number or picks an index that doesn't exist.
